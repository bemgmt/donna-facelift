# Shell Mode - Design Preview Only

This version of DONNA is a **design shell** with no functional features. All API calls, backend connections, and interactive features are disabled.

## What's Disabled

- ✅ ChatWidget - Visual only, no API calls
- ✅ ServiceStatus - Static "Design Preview Mode" status
- ✅ All interface components - Will show static/demo data when opened
- ✅ Voice features - Disabled
- ✅ Email features - Disabled
- ✅ All fetch/API calls - Disabled

## What's Enabled

- ✅ Visual design system (glassmorphic, neon, gradients)
- ✅ Layout and navigation
- ✅ Component styling
- ✅ Animations and transitions
- ✅ Static UI elements

## Deployment

This shell version should deploy successfully on Vercel without requiring:
- Backend APIs
- Database connections
- External service credentials
- WebSocket connections

All components are visual-only and will not attempt to connect to any services.
